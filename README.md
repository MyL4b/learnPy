## üìö Project Overview

This project is designed for learning Python from scratch in a clean, distraction-free environment.  
It was also created out of a personal need: as a computer science student, I constantly wanted to have **Python notes and a working IDE at my fingertips**.

Especially when I was new to Python, having both an IDE and Python notes readily available was incredibly helpful. During my learning phase, I often faced limitations such as **not having an on-the-go IDE**, needing to **set up heavy software**, or being forced to use **online IDEs filled with ads and distractions**.

The goal of this project is to remove those barriers‚Äîallowing users to **tinker, experiment, and practice every day** without wasting time on setup, configuration, or switching between tools. That need for **simplicity, focus, and accessibility** is the main reason this project was created.
